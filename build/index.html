Gulp com TailwindCSS

Atualizado com o Tailwind 2.0, PostCss 8.0+

Referência:

Follow @manjumjn Star Fork Download

Modo de Usar:

Utilize os comandos abaixo para iniciar.

// Para Instalar as Dev Depedencies:

 npm install

// Para iniciar o desenvolvimento

 npm run start

// Para gerar os arquivos minificados,optimizados para produção em servidor

 npm run build

Opções:

Para mudar o caminho dos arquivos e das pastas dist/build configure conforme abaixo:

   

                paths: {
                         root: "./",
                         src: {
                             base: "./src",
                             css: "./src/css",
                             js: "./src/js",
                             img: "./src/img"
                             },
                         dist: {
                             base: "./dist",
                             css: "./dist/css",
                             js: "./dist/js",
                             img: "./dist/img"
                             },
                         build: {
                             base: "./build",
                             css: "./build/css",
                             js: "./build/js",
                             img: "./build/img"
                            }
                        }
